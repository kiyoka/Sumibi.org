

             プロトコル設計メモ

            Protocol design memo
          


1. プロトコルの種類
  - 標準入出力  ... uim,sumibi_soap.cgi,Emacsから使用する。
  - SOAP        ... Emacsとuim等のフレームワークからリモートで使用する。

2. 標準入出力のプロトコル
  - 基本ルール
    入力:
      [関数名]  [パラメータ]
      *  関数名とパラメータの間は1文字以上のスペースとする。

    出力:
      1行目       ( 正常に終了したかどうかを返す )
        ok/error
      2行目以降   ( 関数の結果を返す。複数行の場合も有りえる )

      *  プログラムの終了が出力の終わりとする。


3. メソッド
  doSumibiConvertSexp
    入力引数
      query  ... 変換ソース文字列
      sumi   ... 辞書の名前
    結果
      変換結果をS式で返す。(utf-8)

  doSumibiConvert
    入力引数
      query  ... 変換ソース文字列
      sumi   ... 辞書の名前
    結果
      変換結果を配列で返す。(utf-8)
